# Database-Project
This project implements a University Database Management System using SQL. The database is designed using a normalized relational schema and an Entity Relationship Diagram (ERD) to manage academic information efficiently.
It includes tables for Departments, Staff, Courses, Instructors, Students, Sections, and Enrollment, with proper primary and foreign key relationships to maintain data integrity.
The project demonstrates database design, normalization, table creation, and data insertion using SQL queries, simulating a basic system for managing university academic records.
